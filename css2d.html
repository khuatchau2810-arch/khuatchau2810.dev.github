CSS 2D Transform

Di chuyển chuột vào phần item để xem hiệu ứng

1) translate(x,y)

Translate
Dịch phần tử theo mục X, Y. Ví dụ tranform: translate(50px, 30px)

2)rotate(angle)

Rotate
Xoay quanh tâm: transform: rotate(45deg)

3) scale(sx[, sy])

Scale
Phóng/thu: transform: scale(1.4)

4) skew(ax[,ay])

skew
Ngiêng theo trục x và y

5) Kết hợp nhiều transform

Combo
Ví dụ: translate(...) rotate(...) scale(...) skew(...)

6) matrix(a,b,c,d,tx,ty)

matrix
Ma trận 2D chp phép kết hợp các biến đổi trong một lệnh. Ở đây gần tương đương rotate(20deg) + scale(1.126)
